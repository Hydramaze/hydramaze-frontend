Environment Configuration

To get started in the Artificial Intelligence programming you will need to setup an environment to properly work. In this section we will show you how to do that.

It’s highly recommended that you use a Unix based operational system, because some libraries and tools may not work in different platforms. Straight to the point, let’s get to work.

Linux (Ubuntu)

Python

Most of the recent Linux distributions already come with some version of Python installed, to check the version open the Linux terminal (Ctrl + Shift + t) and execute the following command:

python --version

If the version is equal or greater than 2.7 we’re all set, if you have some older version of Python installed the command below should do the trick:

sudo apt-get update

If you don’t have any version installed you’re really unlucky it’s recommended that you read this.

Python tools and libraries

Okay, we already have Python installed, but that alone don’t do too much, so we will install some other tools which is very helpful.

  • pip

    First of all we will need pip to make the download of dependencies easier. To install pip open the linux terminal and execute the command:

    sudo apt-get install python-pip

  • Every other library

    With pip installed other libraries will be even easier to install, these ones should be enough for you now:

    pip install --user numpy scipy matplotlib ipython pandas sympy nose

    Executing the previous command you will install the scikit-learn dependencies and some other cool stuff. Once that's done, we can go further and install the scikit-learn library:

    pip install -U scikit-learn

Congratulations, your environment is ready to use. =)
